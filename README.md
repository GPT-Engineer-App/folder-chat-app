# folder-chat-app

A streamlit app that uses langchain cromadb sqlite3 and downloaded models (wisper, llama3, llava). In this app users can create folders and upload files to them after that they can open those folder to chat with the files in that folder

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/folder-chat-app.git
cd folder-chat-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
